\# Keycap Sticker Builder



(Я полностью осознаю что данное ПО сырое и кривое. Но оно меня полностью устраивает. Можете полностью переделать под свои нужды. Также принимаю предложение по доработке)



Конструктор кастомных наклеек на клавиатуру с редактором трафаретов.



Возможности:

\- Выбор типа клавиатуры (100%(Кривая), 80%, 60%, кастомная) или загрузка собственной раскладки, созданной в редакторе трафаретов.

\- Загрузка изображений на отдельные клавиши, общий фон клавиш, фон страницы.

\- Настройка размеров клавиш в миллиметрах, внутренних отступов, смещений, режимов вписывания (Cover, Contain, Fill).

\- Плашка с текстом, водяной знак «Сделано на сайте key.rn4.ru».

\- Экспорт в PNG, JPEG, PDF с выбором DPI.

\- Сохранение/загрузка шаблонов .ksticker.

\- Редактор трафаретов (отдельная страница) с сеткой, привязкой к объектам и сетке, перетаскиванием мышью, горячими клавишами, множественным выделением, копированием/вставкой, отступами и фоновым изображением.

\- Защита редактора паролем (HTTP Basic Auth).

\- Поддержка русского и английского языков.

\- Адаптация под мобильные устройства (сенсорные жесты).



\## Установка (Debian 12)



1\. Установите системные зависимости:

&#x20;  ```bash

&#x20;  sudo apt update \&\& sudo apt install -y python3 python3-venv python3-pip nginx

Создайте каталог проекта и скопируйте файлы:



bash

sudo mkdir -p /opt/sticker\_builder/static

sudo mkdir -p /opt/sticker\_builder/layouts

sudo chown -R $USER:$USER /opt/sticker\_builder

Поместите в /opt/sticker\_builder/:



app.py



requirements.txt



в /opt/sticker\_builder/static/ файлы index.html и editor.html



Создайте виртуальное окружение и установите Python-зависимости:



bash

cd /opt/sticker\_builder

python3 -m venv venv

source venv/bin/activate

pip install -r requirements.txt

deactivate

Настройте systemd-сервис. Создайте файл /etc/systemd/system/sticker\_builder.service:



text

\[Unit]

Description=Sticker Builder Flask App

After=network.target



\[Service]

User=www-data

Group=www-data

WorkingDirectory=/opt/sticker\_builder

Environment="PATH=/opt/sticker\_builder/venv/bin"

Environment="TMPDIR=/tmp"

Environment="HOME=/tmp"

ExecStart=/opt/sticker\_builder/venv/bin/gunicorn --workers 2 --bind 0.0.0.0:5000 --worker-tmp-dir /tmp app:app

Restart=always



\[Install]

WantedBy=multi-user.target

Затем:



bash

sudo systemctl daemon-reload

sudo systemctl enable sticker\_builder

sudo systemctl start sticker\_builder

Настройте Nginx для проксирования на порт 5000 (конфигурация как обычно, listen 80, proxy\_pass http://127.0.0.1:5000). При необходимости увеличьте client\_max\_body\_size.



Проверьте права на папку layouts:



bash

sudo chown -R www-data:www-data /opt/sticker\_builder/layouts

sudo chmod 775 /opt/sticker\_builder/layouts

Конструктор будет доступен по адресу http://<ваш\_сервер>/.

Использование

Основной конструктор

Выберите тип клавиатуры или загрузите свою раскладку (если она была сохранена в редакторе трафаретов).



Настройте размер клавиши (1u), фон страницы, общий фон клавиш, плашку.



Кликните по клавише на макете и загрузите картинку.



Правый клик (или долгий тап) открывает индивидуальные настройки для клавиши.



Настройте DPI и формат, затем нажмите «Экспортировать».



Редактор трафаретов

Перейдите на /editor и войдите с логином и паролем.



Включите сетку (G) и привязку к сетке (H). Задайте шаг сетки по X и Y отдельно.



Добавляйте клавиши (N) и отступы (горячие клавиши 1–6 и 0).



Перетаскивайте клавиши мышью, меняйте размеры за уголки, используйте множественное выделение (Shift+Click), копирование/вставку (Ctrl+C/V).



Сохраните раскладку (Ctrl+S). Она появится в выпадающем списке основного конструктора.



Макеты созданные в редакторе появляются в категории "Тип клавиатуры"



Горячие клавиши редактора

N – добавить клавишу



Delete – удалить выделенные объекты



Escape – снять выделение / отменить режим добавления



Стрелки – перемещать выделенные объекты



Shift+Click – множественное выделение



Ctrl+C / Ctrl+V – копировать / вставить



Ctrl+Z / Ctrl+Y – отменить / повторить



Ctrl+S – сохранить раскладку



G – показать/скрыть сетку



H – включить/выключить привязку к сетке



1-6 – вставить отступ (2/5/7/8/13/3.5 мм)



0 – открыть диалог своего отступа
